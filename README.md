# 首都羽翼（Capital Wings）

首都羽翼（Capital Wings）是一个关注北京观鸟与生态保护的微信小程序。旨在通过科普、观鸟记录、数据分享与互动，提升公众对北京地区鸟类及城市生态环境保护的认知和参与度。

## 项目背景

本项目面向北京地区的观鸟爱好者与生态保护关注者，聚合北京常见观鸟地点与鸟种信息，帮助用户认识身边的鸟类、了解它们的栖息地和习性，并鼓励大家走出户外、记录自然。

## 主要功能

- **观鸟地图**：在北京地图上查看 25 个观鸟点位的分布与生态信息。
- **为你推荐**：首页每天展示一种鸟，了解其外观、习性、出现时间与分布。
- **当季常见鸟类**：首页"为你推荐"下方的轮播卡片，配图展示当季可观察的代表性鸟种，左右箭头可切换。
- **鸟类图鉴**：浏览本地常见与代表性鸟种以及各生态区信息，可通过顶部"按拼音 / 按生态区"切换分类方式。
- **互动与数据分享**：浏览观测点与鸟种数据，共建鸟类资料库。

## 技术栈

- **前端**：微信小程序原生框架（WXML、WXSS、JavaScript）
- **后端**：云函数 / Serverless（如微信云开发环境或第三方云服务）
- **数据库**：云开发数据库（如微信云开发）
- **地图服务**：腾讯地图API 或 高德地图API

## 项目结构

> 以下仅列出主要目录与关键文件。

```plaintext
capital-wings/
├── pages/                    # 页面文件
│   ├── home/                 # 项目总览页（观鸟地图、为你推荐和当季常见鸟类轮播）
│   ├── map-detail/           # 全屏地图页（点击首页"全屏"按钮进入）
│   ├── birds/                # 鸟类图鉴页（按拼音 / 按生态区 切换分类）
│   ├── profile/              # 个人中心页
│   └── settings/             # 设置页
├── data/                     # 数据层
│   └── spots/                # 观鸟点位数据（含 markers 转换函数）
├── assets/                   # 静态图片资源
│   ├── birds/season/         # 当季常见鸟类轮播图片（建议使用 JPG）
│   └── ecosystems/           # 生态区封面图（建议使用 JPG）
├── custom-tab-bar/           # 自定义 TabBar 组件
├── app.js                    # 小程序入口
├── app.json                  # 全局配置
├── app.wxss                  # 全局样式
└── theme.json                # 主题配置
```

## 快速开始

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/XINGHAI183/capital-wings.git
   ```
2. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)，选择"导入项目"，将仓库根目录（即 `app.json` 所在目录）作为项目根目录导入。
3. 在项目详情中填入 AppID（首次使用可用测试号），编译即可在首页看到 25 个观鸟点位的地图与"当季常见鸟类"轮播。
4. 如需云端功能，再配置云开发环境。
