# eBird 鸟类保护小程序

eBird 是一个关注北京观鸟与生态保护的微信小程序。旨在通过科普、观鸟记录、数据分享与互动，提升公众对北京地区鸟类及城市生态环境保护的认知和参与度。

## 项目背景

本项目面向北京地区的观鸟爱好者与生态保护关注者，聚合北京常见观鸟地点与鸟种信息，帮助用户认识身边的鸟类、了解它们的栖息地和习性，并鼓励大家走出户外、记录自然。

## 主要功能

- **观鸟地图**：在北京地图上查看 25 个观鸟点位的分布与生态信息。
- **每日一鸟**：首页每天展示一种鸟，了解其外观、习性、出现时间与分布。
- **当季常见鸟类**：首页"每日一鸟"下方的轮播卡片，配图展示当季可观察的代表性鸟种，左右箭头可切换。
- **鸟类图鉴**：浏览本地常见与代表性鸟种，了解外观、习性、出现时间与分布。
- **生态介绍**：包含生态区地图，查看不同栖息地类型（湿地、森林、水库、公园）及其代表物种。
- **互动与数据分享**：浏览观测点与鸟种数据，共建鸟类资料库。

## 技术栈

- **前端**：微信小程序原生框架（WXML、WXSS、JavaScript）
- **后端**：云函数 / Serverless（如微信云开发环境或第三方云服务）
- **数据库**：云开发数据库（如微信云开发）
- **地图服务**：腾讯地图API 或 高德地图API

## 项目结构

> 以下仅列出主要目录与关键文件。

```plaintext
eBird/
├── pages/                    # 页面文件
│   ├── home/                 # 项目总览页（观鸟地图、每日一鸟和当季常见鸟类轮播）
│   ├── map-detail/           # 全屏地图页（点击首页"全屏"按钮进入）

│   ├── birds/                # 鸟类图鉴页
│   ├── profile/              # 个人中心页
│   └── settings/             # 设置页
├── data/                     # 数据层
│   └── spots/                # 观鸟点位数据（含 markers 转换函数）
├── custom-tab-bar/           # 自定义 TabBar 组件
├── app.js                    # 小程序入口
├── app.json                  # 全局配置
├── app.wxss                  # 全局样式
└── theme.json                # 主题配置
```

## 快速开始

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/XINGHAI183/eBird.git
   ```
2. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)，选择导入项目。
3. 配置云开发环境（如需使用云端功能）。
4. 根据需求完善项目配置，预览并上传代码。
