# 首都羽翼（Capital Wings）

首都羽翼（Capital Wings）是一个关注北京观鸟与生态保护的微信小程序。旨在通过科普、观鸟记录、数据分享与互动，提升公众对北京地区鸟类及城市生态环境保护的认知和参与度。

## 项目背景

本项目面向北京地区的观鸟爱好者与生态保护关注者，聚合北京常见观鸟地点与鸟种信息，帮助用户认识身边的鸟类、了解它们的栖息地和习性，并鼓励大家走出户外、记录自然。

## 主要功能

- **观鸟地图**：在北京地图上查看 25 个观鸟点位的分布与生态信息。
- **为你推荐**：首页每天展示一种鸟，了解其外观、习性、出现时间与分布。
- **当季常见鸟类**：首页轮播卡片，配图展示当季代表性鸟种，支持左右切换。
- **鸟类图鉴**：浏览本地鸟种及各生态区信息，可通过顶部"按拼音 / 按生态区"切换分类方式；点击鸟类卡片弹出详情弹窗，包含图集与简介卡片。
- **互动与数据分享**：浏览观测点与鸟种数据，共建鸟类资料库。
- **深色模式**：全页面支持跟随系统主题自动切换。

## 技术栈

- **前端**：微信小程序原生框架（WXML、WXSS、JavaScript）
- **后端**：云函数 / Serverless
- **地图服务**：腾讯地图 API / 高德地图 API

## 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/XINGHAI183/capital-wings.git
   ```
2. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)，导入项目根目录（即 `app.json` 所在目录）。
3. 填入 AppID（首次可用测试号），编译即可运行。
4. 如需云端功能，请另行配置云开发环境。
